Haskell Functor vs \(\Hask\) Functor

Functor for Haskell language is a type F which belong to the type class Functor

It must therefore implement an fmap function.

The couple (F,fmap) is then a real functor in the categorical sense for \(\Hask\) if

for any x :: F a:

  • fmap id x = x
  • fmap (f.g) x= (fmap f . fmap g) x
